# Cekirdek Chat'i bu bilgisayarda çalıştırma

1. VS Code'da bu klasörü açın.
2. Run and Debug paneline gidin.
3. "Run Frontend Dev Server" seçin ve başlatın.
4. Ardından "Run Cekirdek Chat" ile tarayıcıda açın.

Varsayılan adres:
- http://localhost:8080

Eğer port meşgulse, önce çalışan sunucuyu durdurun veya farklı port kullanın.
